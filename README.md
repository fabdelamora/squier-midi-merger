# MIDI channel merger

Standalone **Raspberry Pi Pico** + **MicroPython** box: merges Rock Band 3 **Squier** (and similar) **MIDI channels 1–6** onto **one output channel** for synths that only listen on a single channel (e.g. Moog Mother-32).

- **Hardware / BOM / pins:** [HARDWARE.md](HARDWARE.md)  
- **Wiring diagram:** [docs/WIRING.md](docs/WIRING.md)  
- **Firmware & flashing:** [docs/SOFTWARE.md](docs/SOFTWARE.md)  
- **Glossary:** [docs/GLOSSARY.md](docs/GLOSSARY.md)  

Firmware: [`firmware/main.py`](firmware/main.py) → copy to the Pico as `main.py`.
